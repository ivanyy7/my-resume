# Резюме — Ясинский Иван Иванович

Статическая веб-страница с резюме начинающего разработника (программирование с использованием ИИ-агентов и инструментов вроде Cursor).

**Репозиторий:** [github.com/ivanyy7/my-resume](https://github.com/ivanyy7/my-resume)

## Просмотр

- Откройте файл **`index.html`** в браузере (локально после клонирования).
- Для публичной ссылки опубликуйте репозиторий на **GitHub Pages** или **Vercel** — в корне должен быть `index.html`.

## Кратко о профиле

Начинающий специалист: десктопные приложения для Windows, backend-сервисы, полный цикл от идеи до эксплуатации. Опыт работы с ИИ-инструментами разработки.
